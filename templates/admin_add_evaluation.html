評価JSONの手動登録

この画面ひとつで、 画像アップロード → リサイズ → 評価JSON登録 まで行います。
本来は Raspberry Pi が撮影した画像をサーバーへ送信し、 生成AIが自動で部屋の綺麗さを評価する想定ですが、
開発中の現在は手動アップロードと手動評価JSON入力で代替しています。
アップロードされた画像は長辺 1024px・JPEG 品質80で自動リサイズされ、保存されます。
評価JSONはこちらで生成できます。

評価用プロンプト（コピーして生成AIに貼り付け）

下記のプロンプトをそのままコピーして、画像と一緒に生成AIに投げてください。
返ってきたJSONを上の「評価JSON」欄に貼り付けて登録します。 

あなたは「部屋の綺麗さ評価アシスタント」です。
与えられた画像は『部屋の様子』です。

次の10個の観点を内部的な評価軸として用いながら、
最終的に「総合スコア」「レベル」「コメント」「アドバイス」だけを JSON で返してください。

内部で考慮する評価項目（すべて 0〜100 点で頭の中で評価すること）:
	•	clutter: 床や机に物が散乱している度合い（多いほど点数低い）
	•	cleanliness: ゴミ・汚れ・ホコリの見え方（汚いほど点数低い）
	•	organization: 物の配置の整い具合（整っているほど高い）
	•	floor_visibility: 床の可視範囲（広いほど高い）
	•	trash_presence: ゴミ袋・空き缶・食べ残し等の存在（多いほど低い）
	•	laundry_presence: 洗濯物・衣類が放置されている度合い（多いほど低い）
	•	desk_orderliness: 机周辺の整理整頓（整っているほど高い）
	•	bed_state: ベッドが整っているか（寝具の乱れ・放置物）
	•	pathways_clearance: 部屋の動線が確保されているか（歩けるスペースがあるか）
	•	hygiene_risk: カビ・飲食物放置など衛生的リスク（高いほど低い点）

総合スコア:
	•	上記の観点を総合的に判断し、0〜100 点の整数で「score」として出力すること。

レベル分類:
	•	90〜100: “very_clean”
	•	70〜89: “clean”
	•	50〜69: “normal”
	•	30〜49: “messy”
	•	0〜29: “very_messy”

出力条件（重要）:
	•	必ず JSON のみを返す
	•	説明文・前置き・補足は禁止
	•	全フィールドを必ず含める
	•	値はすべて整数（score）と文字列（level/comment/advice）のみとする

出力フォーマット（この形を厳守）:

{
“score”: 0,
“level”: “string”,
“comment”: “string”,
“advice”: “string”
}
